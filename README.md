📱 Instagram Influencer Engagement Analysis (Power BI)
An advanced Power BI analytics dashboard designed to evaluate influencer performance and engagement dynamics across Instagram. This project transforms complex social media datasets into actionable insights for marketing teams to optimize influencer selection and campaign ROI.


🚀 Project Overview :-
In the competitive landscape of digital marketing, identifying the right influencer is critical. This project utilizes a comprehensive dataset of Instagram influencers to analyze follower growth, engagement rates, and category-wise performance. The final dashboard provides a multi-faceted view of the influencer economy, enabling data-driven decision-making.


🛠️ Key Technical Features :-
1) Robust Data Modeling: Developed a star-schema inspired data model within Power BI to handle relationships between influencer categories, regions, and performance metrics.

2) DAX (Data Analysis Expressions):
Created Calculated Measures for precise Engagement Rate calculations (Eng. Rate = Eng. / Followers).
Implemented Time-Intelligence functions to track growth trends.
Utilized SWITCH/Conditional logic to categorize influencers by tier (Micro, Macro, Mega) based on follower count.

3) Advanced Data Transformation:
Used Power Query (M-code) for data cleaning, handling null values in engagement metrics, and standardizing category names.
Formatted large numerical values (K/M/B) for better dashboard readability.

4) Dynamic Visualizations:
KPI Cards: To highlight total reach and average engagement across the platform.
Category Analysis: A treemap or bar chart showing which niches (Lifestyle, Tech, Fashion) command the highest engagement.
Top Influencer Rankings: Using filtered slicers to identify top performers by region or niche.


📊 Dashboard Insights :-
1) The Engagement Paradox: Often showing that micro-influencers maintain higher relative engagement rates compared to celebrity-tier accounts.

2) Niche Dominance: Identification of the most "viral" content categories based on average likes and comments per post.

3) Regional Reach: Geographic distribution of top-performing influencers.


📂 Tech Stack
Tool: Power BI Desktop
Data Source: Instagram Influencer Engagement Dataset
Logic: DAX, Power Query (M-code)
Design: Custom Dark-Gradient UI for high-contrast professional aesthetics.
